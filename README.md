# Atlas
App gamificada para museos en Bogotá. Ofrece rutas rápidas, retos multilingües (ES/EN/PT) y una experiencia de "laberinto" con pistas. Captura hallazgos para llenar tu pasaporte virtual con sellos y ganar XP. Diseñada para exploradores de 20 a 40 años que buscan cultura interactiva. ¡Recorre la ciudad y completa tu colección de insignias!
